🛠️ AltImageAi Developer Documentation

Build | Contribute | Debug | Deploy

📌 Project Goal

AltImageAi is a web tool that uses AI to generate alt text for images, helping make the web more accessible. It's built using React, TailwindCSS, and deployed via GitHub Pages.

📁 File Structure

src/
├── components/
│   ├── Header.jsx
│   ├── Footer.jsx
│   ├── ImageUploader.jsx
│   └── AltTextDisplay.jsx
├── pages/
│   └── Home.jsx
├── App.jsx
├── main.jsx
└── thinker.css

🔃 Component Flow

  • User uploads image → preview is shown
  • Alt text is generated (mock or AI later)
  • Displayed in AltTextDisplay.jsx
  • State passed via props or lifted to Home.jsx

🧪 Dev Setup

git clone https://github.com/Slothcoder69/AltImageAi.git
cd AltImageAi
npm install
npm run dev

🚀 Deployment

  • Set homepage in package.json
  • Update vite.config.js with base: '/AltImageAi/'
  • Run: npm run deploy

Live URL: https://Slothcoder69.github.io/AltImageAi
